Allows you to add the Author's name to the Title in your RSS feeds

== Description ==

Do you have multiple authors creating post/pages that would like to show up in all RSS Readers.  This simple plugin changes the title in your RSS feeds to include the Title and then the Author's name.  In the settings area, you choose what the separator is between the Title and Author name and how its spaced.


== Installation ==

1. Upload `AddAuthor.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the Settings page with the reminders you want to show up.
